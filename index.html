Bachelor thesis

Icon based indoor navigation
Research has shown that technical aid for navigation such as Google maps has a negative impact on human navigation skills. Now it gets more common with similar solutions indoors. We developed a solution that navigates the user with the help of a series of icons that repressents landmarks, this forces the user to interact with the environment. The main focus of the thesis was to see if the concept is worth doing more research on.
